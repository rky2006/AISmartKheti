# AISmartKheti 🌱

**AISmartKheti** is an AI-powered smart farming web application that helps farmers cultivate healthy crops, track market prices, and make data-driven decisions.

## Features

1. **🔬 AI-Powered Crop Health Diagnosis** — Upload a photo of your crop to get instant disease, pest, or nutrient deficiency analysis with treatment and prevention recommendations.
2. **🌾 Smart Farming Advisory** — Get personalized, location-specific recommendations on planting schedules, optimal irrigation times, and best pest control practices.
3. **📊 Localized Market Price Tracking** — View current and historical market prices for agricultural commodities (Wheat, Rice, Cotton, Onion, Tomato, etc.) with trend sparklines.
4. **📚 Agronomy Knowledge Base** — Search and read farming tips, best practices, and educational articles on crop management, soil health, water management, and more.
5. **📝 Farm Activity Log** — Record and track daily farm activities including planting, spraying, harvesting, irrigation, and more.
6. **🌐 Multi-language Support** — Available in English, Hindi (हिन्दी), and Marathi (मराठी). Language is selected before login.
7. **🌱 Crop Type Selection** — Choose your crop types from 8 categories: grains, horticulture, vegetables, oilseeds, pulses, cash crops, spices, and forage.
8. **🚜 Farming Method Selection** — Specify your cultivation method: Organic, Chemical/Conventional, Integrated Pest Management (IPM), or Zero Budget Natural Farming.
9. **🌿 Organic Fertilizer Preparation Guides** — Step-by-step instructions for Jeevamrut, Beejamrut, Vermicompost, and Panchgavya.
10. **🧪 Soil & Water Report Analysis** — Upload or manually enter soil/water parameters for AI-powered crop suitability analysis and soil amendment recommendations.

## Tech Stack

- **Frontend:** React 19 + TypeScript
- **Bundler:** Vite
- **Routing:** React Router v7
- **Styling:** CSS (custom design system, responsive)
- **State:** React Context API
- **Storage:** localStorage (no backend required)
